Определение кодировки ресурса /index.php

С помощью методов GET и HEAD получили информацию. Заголовок Accept-Charset запроса HTTP сообщает, какую кодировку может понять клиент. Сервер выбирает один из вариантов и использует его. В Content-Type сервер информирует клиент о выборе.
